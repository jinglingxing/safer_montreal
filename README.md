# safer_montreal


## How to use venv

- create an env
```bash
virtualenv -p /usr/bin/python3 venv
```

- use the env
```bash
source venv/bin/activate
```

- install everything from env
```bash
pip3 install -r requirements.txt
```

- install a package
```
pip3 install ___
pip3 freeze > requirements.txt
```
